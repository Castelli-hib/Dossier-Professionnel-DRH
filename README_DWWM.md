# Dossier Professionnel – Développeur Web et Web Mobile (DWWM)

## Objectif du dépôt
Ce dépôt accompagne le **Dossier Professionnel (DP)** présenté dans le cadre du titre
**Développeur Web et Web Mobile**.

Il regroupe des éléments illustrant les compétences en cours d'acquisition :
- développement front-end
- développement back-end
- structuration d’applications web
- intégration d’interfaces métier sécurisées

---

## Structure du dépôt

- `/Document_README`  
  → Documents explicatifs liés au Dossier Professionnel

- `/Template_Maquettes`  
  → Maquettes et supports de conception de l’interface RH

- `/MySite`  
  → Mini-site JavaScript regroupant plusieurs projets front-end  
  (validation et structuration des fondamentaux JavaScript)

---

## Logique de progression 

1. **Acquisition et validation des fondamentaux JavaScript**
   - Manipulation du DOM
   - Gestion des événements
   - Logique applicative côté client

2. **Structuration front-end**
   - Mini-site JavaScript multi-projets
   - Organisation du code
   - Navigation et cohérence UI

3. **Intégration dans un contexte applicatif métier**
   - Interfaces RH sous Symfony
   - Cohérence front / back
   - Sécurité, rôles et contraintes métier

Objectif  **montée en compétences cohérente et professionnelle**.

---

## Positionnement professionnel

Positionnement comme
- capable de concevoir et intégrer des interfaces utilisateurs
- consciente des contraintes de sécurité et d’architecture
- en capacité de structurer du code maintenable
- Attention portée à l’usage métier et à l’ergonomie
